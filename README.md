# werewolfkill
六位地黄丸专用狼人杀

如何使用
--
0. 安装 Python 3.7 版本及以上
1. pip install -r requirements.txt
2. python main.py
3. 所有玩家访问 Web 服务

目前支持
--
1. 基于https://github.com/pywebio/PyWebIO
3. 目前已经调试逻辑正确的角色有，普通狼人，平民，预言家，添加了猎人和女巫在不同状态下，技能的正确使用问题
4. 推荐人数3，3，3
5. 欢迎有兴趣的玩家修改自己的角色逻辑。
